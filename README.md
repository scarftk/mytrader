# myTrader 按钮功能说明

仅说明面板中每个按钮的当前功能。

<img width="2559" height="1193" alt="image" src="https://github.com/user-attachments/assets/f5940b82-c840-4157-917e-61544fddc47e" />

## 顶部区域

- 一键平仓：平掉当前品种全部持仓。
- 订单单独减仓：对当前选中持仓按百分比减仓（百分比来自右侧输入框）。
- 左侧可选择订单操作

## 平仓功能区

- 平仓50%：当前品种全部持仓减仓 50%。
- 平仓80%：当前品种全部持仓减仓 80%。
- 平空：平掉当前品种全部空单。
- 平多：平掉当前品种全部多单。
- 平盈：平掉当前品种全部盈利持仓。
- 平亏：平掉当前品种全部亏损持仓。
- 一键保本：仅对当前选中持仓执行保本（SL 移到开仓价，TP 保持不变）。



## 下单区

- 空：按当前手数执行市价卖出（可附带 SL/TP）。
- 多：按当前手数执行市价买入（可附带 SL/TP）。
- Bid（绿色报价按钮）：与“空”相同，执行市价卖出。
- Ask（红色报价按钮）：与“多”相同，执行市价买入。

## 相关输入框（非按钮）

- 百分比输入框：给“订单单独减仓”使用。
- 手数输入框：给“空/多/Bid/Ask”下单使用。
- SL/TP 输入框：价格单位，在下单时附带到订单。
